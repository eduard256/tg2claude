# Как управлять устройствами в комнате Вадима

## Освещение

### main-vadim-room-light
**Тип:** Двухклавишный выключатель

**Топик:** `zigbee2mqtt/main-vadim-room-light/set`

**Управление:**
- `state_left` - боковой свет (ON/OFF)
- `state_right` - люстра (ON/OFF)

**Пример:**
```json
{"state_left": "ON"}
{"state_right": "OFF"}
```

### main-vadim-bed-yeelight
**Тип:** RGB подсветка под кроватью

**Топик:** `automation_devices/main-vadim-bed-yeelight/set`

**Управление:**
- `state` - вкл/выкл (ON/OFF)
- `color` - цвет RGB (r, g, b: 0-255)
- `brightness` - яркость (0-100)

**Пример:**
```json
{"state": "ON", "color": {"r": 0, "g": 255, "b": 0}, "brightness": 100}
{"state": "OFF"}
```

## Шторы

### main-vadim-window-curtain
**Тип:** Моторизованная штора

**Топик:** `zigbee2mqtt/main-vadim-window-curtain/set`

**Управление:**
- `position` - позиция (0-100, где 0=закрыто, 100=открыто)
- `state` - команды (OPEN/CLOSE/STOP)

**Пример:**
```json
{"position": 50}
{"state": "STOP"}
```

### main-vadim-bed-button_curtain
**Тип:** Кнопка управления шторами (только чтение)

**Топик:** `zigbee2mqtt/main-vadim-bed-button_curtain`

**События:**
- `single` - одиночное нажатие (шторы вниз)
- `double` - двойное нажатие (шторы вверх)
- `hold` / `release` - длительное нажатие (стоп)

## Автоматизации

### vadim/bed-button-light
**Описание:** Управление светом с кнопки у кровати
- Одиночное - левый свет
- Двойное - правый свет
- Длительное - выключить оба + Yeelight ленту

### main/vadim-bedroom/bed-light-motion
**Описание:** Автоматическое управление светом по датчику движения

### vadim/curtain-button
**Описание:** Управление шторами с кнопки
- Одиночное - вниз
- Двойное - вверх
- Длительное - стоп
